# 🌐 Web-Crafter : Drag-and-Drop Website Builder  

**Web-Crafter** is a simple drag-and-drop web design tool built with **HTML, CSS, JavaScript, and Bootstrap**.  
It allows users to create web pages effortlessly by dragging elements like text, images, buttons, and forms onto a customizable canvas.  

The project is designed with an **intuitive UI**, flexible **CSS styling options**, and smooth **navigation** for a seamless user experience.  

---

## 🚀 Features  

- 🎨 **Drag-and-Drop Interface** – Build web pages visually without coding.  
- 📝 **Text Elements** – Add and edit text blocks on the canvas.  
- 🖼️ **Images** – Insert images with adjustable size and placement.  
- 🔘 **Buttons** – Place interactive buttons for navigation and actions.  
- 📩 **Forms** – Add input fields, text areas, and submit buttons.  
- 🎛️ **Customizable Styling** – Modify fonts, colors, and layouts with ease.  
- 🔄 **Responsive Design** – Powered by Bootstrap for cross-device compatibility.  
- 🧭 **Easy Navigation** – Simple toolbar for selecting and managing elements.  

---

## 🛠️ Technologies Used  

- **HTML5** – Structure of the application.  
- **CSS3** – Styling and customization.  
- **Bootstrap** – Responsive design and prebuilt UI components.  
- **JavaScript** – Core logic for drag-and-drop functionality.
